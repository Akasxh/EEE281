# EE381 - Autumn 2022

## **EE381 - Power Systems**

**Instructor:**  
[S. Sivasubramani](http://www.iitp.ac.in/~siva/), Block 3, IIT P Bihta Campus

**Lectures:**  
Monday: 11-12 R104, Thursday: 9-10 R104 and Friday: 10-11 R104

**Syllabus:**
- Architecture of a power system, Line parameter calculation
- Performance analysis of an AC transmission line
- Load flow analysis
- Economic load dispatch
- Short circuit analysis
- Stability analysis

**Prerequisites:**
1. EE101 - Electrical Sciences
2. EE280 - Electrical Machines

**Text Books:**
1. W. D. Stevenson, Jr., Elements of Power System Analysis, Tata McGraw-Hill, 4e, 2008.

**References:**
1. J. J. Grainger and W. D. Stevenson, Jr., Power System Analysis, Tata McGraw-Hill, 2008.
2. D. P. Kothari and I. J. Nagrath, Modern Power System Analysis, Tata McGraw-Hill, 3e, 2010.
