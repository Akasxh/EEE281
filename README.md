# EE381 -- Power Systems

Course materials for EE381 (Power Systems), Autumn 2022, IIT Patna.

**Instructor:** [S. Sivasubramani](http://www.iitp.ac.in/~siva/), Block 3, IIT Patna Bihta Campus

## Syllabus

- Architecture of a power system, line parameter calculation
- Performance analysis of AC transmission lines
- Load flow analysis
- Economic load dispatch
- Short circuit analysis
- Stability analysis

**Prerequisites:** EE101 (Electrical Sciences), EE280 (Electrical Machines)
